## 工数管理システム
作業の開始時間、終了時間、完了台数を入力することで、一台あたりの作業時間を算出する。
## なぜ開発したのか
繰り返し生産される製品に対し、データを蓄積することにより改善点をつきとめ、業務の効率化をはかる。
